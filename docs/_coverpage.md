> A magical UI generator.

* **开箱即用**，内置React（[Ant Design](https://www.npmjs.com/package/antd)）、Vue（[Element](https://www.npmjs.com/package/element-ui)）实现
* **五大基本组件**，组合创建无限可能
* **基于配置生成界面**，提供命令行工具可输出源码
